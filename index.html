Turing Machine Simulator

How To Use

Information

1. Only used to work on m x alogb / n problems.

2. Only accepts strings "0", "1", and "X".

3. The string "0" represents the number of a variable.

4. The string "1" indicates the variable is negative.

5. The string "X" is used as a separator between variables.

6. The string template is 0mX0nX0a0b.

Writing Input

1. The number of strings "X" must be 3.

2. The string "1" must be followed by the string "0" behind it.

3. The string "0" cannot be in front of the string "1".

Example Valid Input

1. 0000X00X00X0000 for 4 x 2log4 / 2

2. 100000X000X000X00000 for -5 x 3log5 / 3.

3. 1000X1000X00X00000 for -3 x 2log5 / -3.
